# Data Directory

This directory contains runtime data for the HotVideo platform:

- `audio/`: Generated and uploaded audio files
- `browser_profiles/`: Browser automation profiles
- `output/`: Processed video outputs
- `postgres/`: PostgreSQL database files
- `redis/`: Redis cache data
- `thumbnails/`: Video thumbnails
- `videos/`: Source and processed video files
- `web/`: Temporary web assets

These directories are excluded from git via .gitignore.
